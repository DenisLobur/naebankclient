# Naebank Android client
Naebank is a new client bank which has (potentially) all the basic functionality any user is going to need.

# Features
- Login user
- Register user
- Logout user

- Show cards list
- Add card
- Withdraw money
- Top-up money
- Delete card
- Show card's transactions

- Show all user's transactions
